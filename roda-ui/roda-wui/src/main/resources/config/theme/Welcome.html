Welcome to ETERNA!

An open-source digital repository designed for preservation

ETERNA is a long-term digital repository solution that delivers functionalities for all the main functional units of the OAIS reference model. ETERNA is capable of ingesting, managing, and providing access to various types of digital content produced by large corporations and public bodies.

Conforms to open standards
Vendor independent
Scalable
Embedded preservation actions
Authenticity
Support for multiple formats
Copes with the rapid changing nature of technology
Advanced access control
Integration with 3rd party systems
Advanced ingest workflow
Retention and disposal policies
Marketplace

For more information about ETERNA and its main features, please visit www.whitered.se/eterna.
